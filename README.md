# Lag Pac

Arch Linux update manager - delays updates to 14-day intervals, keeps gaming/browser stuff current.

## Installation
```bash
git clone https://github.com/m4c4r0n1n/lag-pac.git
cd lag-pac
./install.sh
```

## Usage

- `pac -U` - System update (14-day schedule)
- `pac -C` - Update Firefox/Steam/gaming
